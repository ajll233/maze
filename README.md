# maze
控制人物走出迷宫
走迷宫：
    数据分析:
        1、定义二维字符数组作为迷宫地图
        2、定义两个变量来记录角色的位置x y
        3、时间：time(NULL)  获取1970-1-1到运行时过了总秒数
    逻辑分析:
        进入死循环：
            1、显示地图
            2、等待获取方向键并处理
                判断前方是不是路" "
                    如果是：
                        1、把原位置" "
                        2、把新位置变成"@"
                        3、跟新角色位置坐标 x y 
            3、判断是否到达出口
                    如果是：程序结束
